# feSvg 🎨

> Create stunning SVG filter effects with a visual node-based editor

**feSvg** is an intuitive visual editor for creating and sharing complex SVG filters. Connect filter nodes like in 3D material editors, preview results in real-time, and share your creations with the community.

![Status](https://img.shields.io/badge/status-in%20development-yellow)

## ✨ Features

- **🔗 Visual Node Editor** - Connect filter components with an intuitive drag-and-drop interface
- **👁️ Real-time Preview** - See your filter effects as you build them
- **📚 Filter Library** - Browse and use pre-built filters from the community
- **🔄 Import & Export** - Parse existing SVG filters or export your creations
- **🎯 Complete Filter Support** - All SVG filter primitives (feBlend, feGaussianBlur, feColorMatrix, and more)
- **☁️ Cloud Sync** - Save and manage your filters online
- **⭐ Share & Discover** - Publish your filters, rate others, and follow creators
